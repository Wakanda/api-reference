Global Scope

The application object available on the global scope represents both the currently running application and the JavaScript context global scope. This means that all properties and methods of the application object are directly accessible from the global scope and vice versa.

In practice, this implies that the following code snippets are equivalent :

var uuid = generateUUID();

And

var uuid = application.generateUUID();

The application object implements the Application interface.
